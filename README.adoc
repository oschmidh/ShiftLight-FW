# ShiftLight Firmware

image::https://github.com/oschmidh/ShiftLight-FW/actions/workflows/build.yml/badge.svg?branch=main[]

## Setup

[source,console]
----
meson setup --cross-file arm-none-eabi.txt build
----

## Compile

[source,console]
----
meson compile -C build
----

## Debug

[source,console]
----
pyocd pack install mspm0c1104
----

[source,console]
----
pyocd load -t mspm0c1104 -O reset_type=hw ./build/app/shiftlight.elf
pyocd commander -t mspm0c1104 -c continue
----
